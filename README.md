# Ecommerce TripleDStore
# Description
  TripleDStore is an ecommerce store that sells fashionable clothes and accessories.
# Stack
  ## Front-End
    HTML, CSS, Javascript.
  ## Back-End
    Python and Django.
# Installation
  1. Create a python virtual environment. I recommend using `virtualenv` to create the virtual environment.
  2. Activate the virtual environment and install the dependencies on the requirements.txt file. To install dependencies `pip install -r requirements.txt`.
  3. Run the django server using the following command `python manage.py runserver`.
  4. On your browser visit this url `http://localhost:3000`
